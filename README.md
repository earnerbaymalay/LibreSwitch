> **"Flip the switch on privacy. Reclaim your device."**

**LibreSwitch** is a Cyberpunk-styled privacy dashboard. It scans your device for proprietary, data-harvesting apps and provides instant FOSS replacements.

## üì± Visuals
The dashboard calculates a **System Integrity Score** based on how many tracking apps you have installed.

*   **Deep Scan:** Detected **22 Threats** on test devices (Chrome, YouTube, TikTok, etc).
*   **The "Nuclear Option":** Uses **Shizuku** to uninstall system bloatware without Root.
*   **Massive Database:** Supports 40+ replacements including NewPipe, Mull, and Signal.
